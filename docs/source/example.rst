Examples
********************************************************************************

To make the first steps easier, two template scripts are available covering the simulation of simple and articulated vehicles. We encourage the users to run and explore the examples :ref:`template-simple` and :ref:`template-articulated`.

Alternatively, for Matlab users familiar with Simulink, two template applications are available for running the models of the package in Simulink. Run and explore the examples :ref:`template-simple-simulink` and :ref:`template-articulated-simulink`.
